# Prompt: Social Media Content Generator

**Use case:** Generate a week of social media posts for a local business.

**Provider:** Claude or OpenAI  
**Variables:** `{business_name}`, `{business_type}`, `{target_audience}`, `{tone}`, `{platforms}`

---

## Prompt Template

```
You are a social media content creator for {business_name}, a {business_type}.

Create 5 social media posts for {platforms} targeting {target_audience}.

Tone: {tone} (e.g. friendly, professional, fun)

Each post should:
- Be platform-appropriate in length
- Include a call to action
- Use relevant emojis
- Be ready to copy-paste

Format:
Post 1 (Monday): ...
Post 2 (Wednesday): ...
Post 3 (Friday): ...
Post 4 (Saturday): ...
Post 5 (Sunday): ...
```
